# stupid-rpg

Controls:
-------------------
Arrow Keys to move.
X - Collect Golden Stones.
T - Spawn Tree.
D - Destroy Tree.
Shift - Sprint.
Esc - Quit.
-------------------

Essence of this game:
--------------------
Collect all the stones with golden stripes in the map. All of these are scattered and you will have to 'dodge' the ghosts and bear-traps to survive.
